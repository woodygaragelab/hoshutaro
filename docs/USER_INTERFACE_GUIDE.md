# ユーザーインターフェースガイド

## 概要

HOSHUTAROは保全管理のためのExcelライクなインターフェースを提供し、直感的なナビゲーションと包括的な編集機能を備えています。本ガイドでは全てのUIコンポーネントとユーザー操作について説明します。

## メインインターフェースレイアウト

## モダンヘッダーコンポーネント

### 1. 検索・フィルタリング

#### 機器検索

- **配置**: ヘッダー左側
- **機能**: リアルタイム機器名フィルタリング
- **使用方法**: 機器名またはTAG No.を入力してフィルタ
- **ショートカット**: `Ctrl+F` で検索ボックスにフォーカス

#### 階層フィルタ

- **配置**: フィルタドロップダウンメニュー
- **レベル**: 製油所、エリア、ユニット
- **機能**: 階層レベルでフィルタリング
- **使用方法**: レベルを選択して該当階層の機器のみ表示

#### 詳細フィルタ

- **アクセス**: フィルタメニュー → 詳細
- **機能**:
  - 複数条件検索（AND / OR）
  - 日付範囲フィルタ
  - 作業分類フィルタ
  - 費用範囲フィルタ
  - フィルタ条件の保存・管理

### 2. 表示モード切り替え

#### ステータス / 費用 切り替え

- **配置**: ヘッダー中央
- **選択肢**:
  - **ステータス表示**: 保全実施状況を表示（○●△×）
  - **費用表示**: 費用データと予算情報を表示
- **使用方法**: クリックでビューを切り替え

#### 時間スケール選択

- **配置**: ビューモード切り替えの隣
- **選択肢**: 日次、週次、月次、年次
- **機能**: 時間集約の単位を変更
- **使用方法**: スケールを選択して列のグルーピングを変更

### 3. アクションメニュー

#### データ操作

- **データエクスポート**: 現在のデータをJSONでダウンロード
- **データインポート**: JSONファイルをアップロードしてデータを置き換え
- **データリセット**: 初期サンプルデータに復元

#### 年度管理

- **年度追加**: 新しい年度列を追加
- **年度削除**: 年度列を削除（バリデーション付き）

#### 表示設定

- **TAG No.表示**: 機器IDの表示切り替え
- **サイクル表示**: 保全サイクルの表示切り替え
- **仕様表示**: 機器仕様の表示切り替え

## 拡張メンテナンスグリッド

### 1. グリッド構造

#### 機器ベースモード（タブ名：機器）

```
設備所在             │ 機器名           │ 1月 │ 2月 │ 3月 │
第一製油所           │                 │     │     │     │
├─ Aエリア           │                 │     │     │     │
│  ├─ 原油蒸留ユニット │                 │     │     │     │
│  │  └─ P-101      │ 原油供給ポンプ   │  ○  │  ●  │ ◎(2)│ ← 全WorkOrderLineを集約
```

- **行**: Asset（HierarchyPath階層でグルーピング）
- **セル**: 当該Asset×日付の全WorkOrderLineのステータスを集約
- **ダブルクリック**: 当該Asset×日付に関連するWorkOrderのWorkOrderLineを一覧表示

#### 作業ベースモード（タブ名：作業）

```
設備所在             │ 作業名                          │ 1月 │ 2月 │ 3月 │
第一製油所           │                                │     │     │     │
├─ Aエリア           │                                │     │     │     │
│  ├─ 原油蒸留ユニット │                                │     │     │     │
│  │  └─ P-101      │ 原油供給ポンプ                   │     │     │     │
│  │      ├─         │ 原油供給ポンプ分解清掃           │  ○  │  ●  │     │
│  │      └─         │ 原油供給ポンプ電動機ベアリング交換 │     │     │  ○  │
```

- **行**: 階層 → Asset → WorkOrderLine（3階層）
- **セル**: 特定WorkOrderLine（1 Task × 1 Asset）のステータス
- **ダブルクリック**: 当該Asset×日付に関連するWorkOrderのWorkOrderLineを一覧表示

### 2. ビジュアルインジケーター

#### ステータス記号

- **○** 計画済み（白丸）
- **●** 実施済み（黒丸）
- **◎** 計画済み＋実施済み（二重丸）
- **◎(n)** 複数WorkOrderLine集約（括弧内に件数）

#### カラーコード

- **緑**: 予定通り完了
- **黄**: 一部完了または遅延
- **赤**: 期限超過またはキャンセル
- **青**: 将来の計画
- **灰**: スケジュールなし

### 3. セル操作

#### 単一セル編集

1. セルを**クリック**して選択
2. **ダブルクリック** または **F2** で編集開始
3. **Enter** で保存、**Esc** でキャンセル
4. **Tab** で次のセルに移動
5. **Shift+Tab** で前のセルに移動

#### 複数セル選択

1. **クリック＆ドラッグ** で範囲選択
2. **Ctrl+クリック** で複数セル選択
3. **Shift+クリック** で選択範囲を拡張
4. **Ctrl+A** で表示中の全セルを選択

#### コピー＆ペースト操作

1. **Ctrl+C** で選択セルをコピー
2. **Ctrl+V** で選択セルにペースト
3. **対応データ**: ステータス値、費用データ、メモ
4. **バリデーション**: 自動データ型チェック

## ダイアログコンポーネント

### 1. WorkOrderLine一覧ダイアログ

#### アクセス方法

- **機器ベースモード**: セルをダブルクリック
- **作業ベースモード**: セルをダブルクリック
- **キーボード**: セル選択後 **Enter** キー

#### 機能

- **WorkOrderLine一覧**: 当該Asset×日付に関連する全WorkOrderLineを表示
- **WO名 / 作業名**: WorkOrder名と具体的なTask名を並列表示
- **スケジュール編集**: 各日付の計画 / 実績ステータス
- **費用入力**: 計画費用と実績費用
- **工数入力**: 作業工数（人時）
- **WorkOrderLine追加**: 新しいWorkOrderLineの作成

#### インターフェース

```
┌───────────────────────────────────────────────────────────────┐
│ P-101 原油供給ポンプ — 2024年1月 WorkOrderLine一覧            │
├───────────────────────────────────────────────────────────────┤
│ WO                        │ 作業                │ 計画 │ 実績 │
│ ボイラー設備年次点検整備   │ ドラム分解清掃      │  ○   │  ●  │
│ ボイラー設備年次点検整備   │ 弁体摺り合わせ      │  ○   │     │
│ タービン建設OH             │ ベアリング交換      │  ○   │  ●  │
│                                                               │
│ [WorkOrderLine追加]  [Cancel] [保存]                           │
└───────────────────────────────────────────────────────────────┘
```

### 2. 階層編集ダイアログ

#### 目的

- 設備所在の階層構造を編集
- 階層レベルの値を変更
- 機器の配置を再編成

#### 機能

- **レベル選択**: 編集する階層レベルを選択
- **値管理**: 階層値の追加、編集、削除
- **機器割り当て**: レベル間での機器移動
- **バリデーション**: 不正な階層構造の防止

### 3. 機器再配置ダイアログ

#### 目的

- 機器を別の階層位置に移動
- 一括機器再配置
- 移動時のデータ整合性維持

#### 手順

1. **機器選択**: 移動する機器を選択
2. **移動先選択**: 新しい階層パスを選択
3. **移動検証**: 競合チェック
4. **確認**: 確認後に再配置を適用

### 4. ステータス選択ダイアログ

#### クイックアクセス

- セルを**右クリック**でコンテキストメニュー
- **キーボードショートカット**: 選択セルで **Space**

#### 選択肢

- **計画** (○): 計画保全としてマーク
- **実績** (●): 完了保全としてマーク
- **計画＋実績** (◎): 計画＋完了としてマーク
- **クリア**: 全ステータスを削除

### 5. 費用入力ダイアログ

#### 機能

- **計画費用**: 計画保全費用の入力
- **実績費用**: 実績保全費用の入力
- **通貨**: 自動円（¥）フォーマット
- **バリデーション**: 数値入力のバリデーション
- **自動ステータス**: 費用入力時にステータスを自動設定

### 6. 仕様編集ダイアログ

#### 目的

- 機器仕様の編集
- 仕様の表示順管理
- 仕様フィールドの追加 / 削除

#### インターフェース

```
┌──────────────────────────────────────────────────┐
│ 機器仕様 - P-101 原油供給ポンプ                   │
├──────────────────────────────────────────────────┤
│ 仕様項目         │ 値           │ 順序 │ 操作   │
│ メーカー         │ A社          │   1  │ [↑↓]  │
│ 型式            │ ABC-123      │   2  │ [↑↓]  │
│ 容量            │ 100L/min     │   3  │ [↑↓]  │
│                                                  │
│ [仕様追加] [保存] [キャンセル]                      │
└──────────────────────────────────────────────────┘
```

## ビジュアルデザイン

### カラースキーム

- **プライマリ**: 青 (#1976d2) — ヘッダーとアクション
- **成功**: 緑 (#4caf50) — 完了項目
- **警告**: オレンジ (#ff9800) — 一部完了 / 遅延項目
- **エラー**: 赤 (#f44336) — キャンセル / 期限超過項目
- **情報**: 水色 (#2196f3) — 計画項目
- **ニュートラル**: 灰 (#757575) — 非アクティブ項目

### タイポグラフィ

- **ヘッダー**: Roboto Bold, 16px
- **本文**: Roboto Regular, 14px
- **セルテキスト**: Roboto Regular, 12px
- **ボタン**: Roboto Medium, 14px

### スペーシング

- **グリッドセル**: 8px パディング
- **ダイアログマージン**: 16px
- **ボタン間隔**: 8px
- **セクション間隔**: 24px

## レスポンシブ動作

### デスクトップ（1280px以上）

- 全機能が利用可能
- 全列を表示
- 完全なツールバー
- マルチパネルレイアウト

### タブレット（768px〜1279px）

- 折りたたみ可能なサイドバー
- グリッドの水平スクロール
- 簡略化されたツールバー
- タッチ最適化コントロール

### モバイル（767px以下）

- **非対応** — デスクトップ専用アプリケーション
- デスクトップ版へのリダイレクトメッセージを表示

## カスタマイズオプション

### 表示設定

- **列の表示/非表示**: 特定の列を表示切り替え
- **行グルーピング**: 階層レベルの展開/折りたたみ
- **セルフォーマット**: 数値形式、日付形式
- **テーマ選択**: ライト/ダークモード（将来対応）

### ユーザー設定

- **デフォルト表示モード**: ステータスまたは費用
- **デフォルト時間スケール**: 日次、週次、月次、年次
- **自動保存間隔**: 自動保存の頻度
- **通知設定**: 成功/エラーメッセージの設定

本UIガイドにより、HOSHUTAROの全機能を効果的に操作するための指針を提供します。
