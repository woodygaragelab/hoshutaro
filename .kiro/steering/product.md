# Product Overview

## HOSHUTARO (保全管理システム)

HOSHUTARO is a maintenance management system featuring an Excel-like grid interface for equipment maintenance planning and tracking. The system displays maintenance schedules in a "star chart" format (星取表) where users can track planned vs actual maintenance activities across different time periods.

### Core Features
- **Enhanced Maintenance Grid**: Excel-like interface with virtual scrolling for large datasets
- **Hierarchical Equipment Management**: Multi-level equipment categorization and filtering
- **Dual View Modes**: Status tracking (planned/actual) and cost management
- **Desktop-Optimized Design**: Optimized for desktop use
- **AI Assistant Panel**: Mock AI integration for future enhancement
- **Data Import/Export**: JSON and Excel file support
- **Real-time Search**: Equipment filtering with hierarchical navigation

### Target Users
Maintenance managers and technicians who need to plan, track, and analyze equipment maintenance schedules across industrial facilities.

### Key Value Proposition
Transforms traditional paper-based or basic digital maintenance tracking into a modern, responsive, Excel-like experience with advanced filtering and AI-assisted planning capabilities.
